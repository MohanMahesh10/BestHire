## ✨ Key Features

### Core Functionality
- ✅ AI Resume Parsing (PDF, DOCX, TXT)
- ✅ Smart Candidate Matching (ML-based)
- ✅ Sentiment Analysis
- ✅ Real-time Analytics Dashboard
- ✅ Dual Processing Modes (Local ML + Gemini API)
